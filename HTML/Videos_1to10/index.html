Imágenes en HTML

La etiqueta img es la que se encarga de cargar las imágenes dentro de las webs html, y sus atributos son:

  • alt: Carga un texto si falla la imágen durante la carga.
  • title: Título que se le asigna a la imágen.
  • src: Sirve para indicar la ruta de la imágen para que se cargue en la web.
  • width: Sirve para indicar la anchura de la imágen.
  • height: Sirve para inidicar la altura de la imágen.

Tablas en HTML

Hace años se usaban las tablas para maquetar las web en HTML actualmente esto se hace a través de CSS con el sistema de cajas.
Actualmente las tablas se usan solamente para introducir información de forma organizada.

La etiqueta table es la que se encarga de representar las tablas y sus etiquetas son:

  • Etiqueta ENCABEZADO: tT: Se encarga de representar la fila del encabezado en negrita.
  • Etiqueta FILAS: tr: Se encarga de representar una fila.
  • Etiqueta COLUMNAS td: Se encarga de representar una columna.
  • Atributo bordesborder: Dibuja los bordes de la tabla.
  • Atributo expandircolspan: Sirve para expandir las columnas.
  • height: Sirve para inidicar la altura de la imágen.

Formularios en HTML

La etitueta que se usa para crear un formulario en html es: form, la etiqueta form cuneta con bastantes atributos los mas destacados son:

  • Action: para indicar donde queremos enviar el formulario.
  • method: para indicar que método HTTP queremos utilizar.
    ara incluir una etiqueta que de una descripción al campo que vamos a añadir a nuestro formulario debemos usar la etiqueta label. Esta etiqueta se relacionara con el campo del formulario que deseemos si introducimos el atributo for simplemente introducimos el name de nuestra input.
    Para incluir un campo de tipo texto usamos la etiqueta input y le decimos mediante el atributo type que es de tipo texto, con el atributo name indicamos el nombre para que un Script exterior acceda al campo, y el atributo id le damos un identificador único en este documento, para posteriormente hacer referencia a él, con el atributo placeholder aparecerá autorelleno el campo del formulario, y cuando escribamos encima automáticamente se borra.
    Para incluir un campo de tipo area de texto usamos la etiqueta texarea, la cual cuenta con los mismos atributos de name y descripcion entre otros.
    Para inclir una pequeña lista de seleccion con botones de tipo circulares usamos la etiqueta input type="radio" y de igual manera le asignamos un atributo name para poder vincularlr y luego el atributo value el cual define el valor que enviara esté elemento.
    Para incluir campos de selección usamos la etiqueta select utilizaremos los atributos name e id para identificar el campo del formulario. Dentro de nuestra etiqueta select debemos usar la etiqueta option la cual está definda mediante el atributo valepar determinar su valor. Podemos usar tantas opciones como deseemos.
    Por último incluimos un boton para enviar nuestro formulario con la etiqueta input type="submit" y le asignamos un atributo value para definir el valor que queremos que aparezca dentro del botón. Si no indicamos nada el formulario lo guarda en la propia web mediante un GET se puede revisar los valores en la barra de dirección que aparecen al enviar el formulario.
